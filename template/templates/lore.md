## Notes

> [!tip]- Property Guide
> - **icon:** Iconize glyph shown before links to this note (default `LiScroll`). The portrait generator overrides this with the entity's image once art exists.
> - **portrait:** wikilink to this entity's image (optional; lore often has none); powers the gallery card view (and, later, inline link icons). Keep in sync with the infobox image.
> - **category:** `technology`, `history`, `culture`, `geography`, `politics`, `religion`, `science`, or `event`
> - **status:** `draft` → `ready` → `revealed` → `retired`
> - **canon:** `true` (canonical to the universe) or `false` (homebrew, what-if)
> - **source:** list of wikilinks to source notes (e.g., `["[[chapter-01]]"]`)
> - **clue:** `true` if this lore entry is a clue
> - **links-to-secret:** wikilink to the secret this clue connects to
> - **found-at:** wikilink to the location where this lore can be discovered
